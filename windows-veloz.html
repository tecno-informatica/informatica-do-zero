DEIXANDO WINDOWS MAIS RÁPIDO E LEVE.

O Windows foi lançado em 1985, mas até hoje, muitos usuários que utilizam o Sistema Operacional não sabem fazer algumas configurações que o mesmo tem para um desempenho melhor.

Todos nós sabemos que de lá para cá, já foi realizado várias atualizações, com mudanças enorme em seu layout e em processamento e desempenho.

Então, irei disponibilizar algumas dicas e configurações para deixar seu sistema operacional Windows 10, mais rápido, com o melhor desempenho, mesmo que sua máquina seja com pouca memória e de baixo processamento.

  1. Desativar Efeitos Visuais Desnecessário

    • Nessa primeira dica, iremos mostrar como fazer para deixar seu Windows mais rápido na inicialização, usando a configuração de “Opções de Desempenho”. Veja a ilustração abaixo;

      passo 02

    • AUTOEXPLICATIVO: Primeiramente, você vai até a pesquisa do Windows e digite “Ajustar a aparência e o desempenho do Windows”, basta digitar, ajustar a aparência, em seguida clique na primeira opção. Após isso, irá aparecer a janela de “Opções de Desempenho”, marque a opção, “Ajustar para obter um melhor desempenho”. Nas caixas abaixo, veja o que é mais útil para você, no meu caso, eu marquei somente essas três opções, elas foram o suficiente para deixar uma inicialização mais rápida, sem dá nenhum prejuízo para meu Windows.

  2. Limpeza de Disco

    • AUTOEXPLICATIVO: Como mostra as imagens acima, você vai até Windows Explorer → Este Computador → Disco que está instalado seu sistema operacional (Botão Direito mouse) → Propriedades → Limpeza de Disco → Marque todas opções → Limpar Arquivos do Sistema → Aguarde calcular → Ok → Excluir Arquivos → Aguarde Limpar. Fazendo essa limpeza semanalmente, você terá um sistema mais rápido.

  3. Desativar Aplicativos em Segundo Plano

    • Aplicativos em Segundo Ativado, significa que mesmo que você não esteja usando o aplicativo, mesmo assim você receberá notificação relacionado a ele, como também o mesmo poderá fazer atualizações disponível, fazendo com que seu sistema fique mais comprometido.

      • passo 01

    • AUTOEXPLICATIVO: Para chegar até a opção Desativar Aplicativo em Segundo Plano: Vá até em Configurações → Privacidade → Nas opções da coluna à esquerda, desça até Aplicativos em Segundo Plano. Assim você chegará na imagem acima, com isso, você pode desativar o botão.

  4. Programas - Não Inicializar junto com Windows

    • No Gerenciador de Tarefas, existe uma opção chamada Inicializar, que serve para desabilitar serviço de programas que inicializa junto com sistemas operacional Windows. Veja a ilustração abaixo;

      • passo 01

      passo 02

    • AUTOEXPLICATIVO: Para realizar a configuração ilustrada acima, você precisa ir até a pesquisa do Windows e digitar “Gerenciador de Tarefas”, clicar na primeira opção. Na janela que aparecerá, vá até inicializar. Observe, que nos programas que o status esteja habilitado, significa que aquele programa está inicializando junto com Windows. Para desabilitar, basta clicar em cima do programa e ir até desabilitar. Com isso, você observará que seu sistema irá inicializar muito mais rápido.

  5. Pasta TEMP – Excluir Arquivos/Pastas

    • A pasta TEMP armazena arquivos temporariamente usado pelo seu Windows. Portanto, pode-se excluir todos os arquivos e pastas que estão dentro da pasta TEMP, sem nenhum prejuízo para seu Windows.

      • passo 01

    • AUTOEXPLICATIVO: De acordo com as imagens ilustradas acima, basta segura a tecla do windows + R, digita no espaço a palavra TEMP, você chegará em uma pasta com vários arquivos e pastas, você pode selecionar tudo (Ctrl + A) e apagar todos.

  6. Pasta %TEMP% - Excluir Arquivos/Pastas

    • Essa dica é parecida com anterior, a grande diferença está no símbolo de % que se usa no início e no final da palavra TEMP. Nessa pasta, existe vários arquivos e pastas temporariamente criado pelo Windows, de acordo com seu uso. Pode-se selecionar todos e excluir sem prejuízo para seu Windows.

      • passo 01

    • AUTOEXPLICATIVO: Aqui também, usa os mesmos critérios usados no autoexplicativo anterior, pode fazer do mesmo jeito, não tendo nenhum prejuízo para seu Windows.

  7. Pasta RECENT – Excluir Arquivos/Pastas

    • Outra pasta que armazena muitos arquivos e pastas temporariamente do Windows, nessa você também pode excluir tudo que está dentro dela.

      • passo 01

    • AUTOEXPLICATIVO: De acordo com a imagem acima, você pode acessar essa pasta e excluir todos os arquivos e pastas que estão localizadas dentro dela, você não terá nenhum prejuízo para seu Windows.

  8. Pasta PREFETCH – Excluir Arquivos/Pastas

    • A pasta PREFETCH é aonde são armazenados dados de pré-carregamentos de aplicativos, uma vez cheia, isso faz com que seu Windows fica mais lento, seja na inicialização e durante o uso. Pode excluir todos os arquivos e pasta que fica dentro dela, isso não afetará o desempenho do seu Windows para pior, ao contrário, seu sistema irá processar muito mais rápido com a exclusão de arquivos.

      • passo 01

    OBSERVAÇÂO: Todas essas configurações, você poderá desfazer, caso não queira utilizar mais. Após realizar essas configurações, você deve obrigatoriamente inicializar seu sistema para atualizar. Para somar com essas dicas citadas acima, você também pode manter poucos atalhos na sua área de trabalho, para ter uma inicialização mais rápida ainda. Como também, desinstalar programas inutilizáveis. Existem outras dicas, mas essas citadas acima são as que mais fazem efeitos positivo no seu sistema Windows, pode aplicar como foi mencionado acima, terá uma melhor rapidez e desempenho.

  9. DICA EXTRA:

    • Depois de fazer todas essas configurações citadas acima, mas mesmo assim gostaria que seu Sistema Operacional Windows, processasse e inicializasse muito mais rápido ainda! Então, recomendo a instalação de um SSD. Disco de Estado Sólido (SSD), é uma unidade com tipo de armazenamento mais rápido e silencioso, que armazena dados em um chip de memória que pode ser acessado imediatamente, oferecendo uma durabilidade e confiança para os usuários.
